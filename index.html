fractus by benhardy

Scala-based flame fractal generation

This is something I'm tinkering with to help me learn Scala. It makes pretty pictures. It's super-bleeding edge alpha quality, at the moment, which means it's not particularly user friendly yet, unless your main interest in it is learning Scala and drawing pretty pictures. Once you've built it, there's a "fractus2" script in the source root that you can then run this with by providing a fractal name and optionally pixel dimensions for the image to render. The fractal names are defined in the RuleMap class using a rather lightweight set of expressions. For example, here's the rule for generating a Sierpinski Gasket (ya know, that triangly thinger): "sierpinski" -> Array[Rule]( weight(1) color(RED) also { _ * 0.5 + ( 1.0, 0.0) }, weight(1) color(GREEN) also { _ * 0.5 + (-1.0, -1.0) }, weight(1) color(BLUE) also { _ * 0.5 + (-1.0, 1.0) } ), I wanted to make the rule expressions concise yet readable. Really complex conditional rules are possible as well, though the expressions for these aren't nearly as concise (yet).

Dependencies

Scala, Java, Maven

Install

Download the source and build it with maven2 using the "mvn package" command. Maven will download the scala and other dependencies.
